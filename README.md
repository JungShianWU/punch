# 打卡小工具

一鍵上下班打卡，自動計算預計下班時間與加班起算時間，支援傳送 LINE。

## 使用方式

開啟網頁 → 點「上班打卡」或「下班打卡」→ 選擇「傳送 LINE」或「拷貝」。

**網頁連結：** https://jungshianwu.github.io/punch/

### 加到 iPhone 主畫面

1. 用 Safari 開啟上方連結
2. 點分享按鈕 →「加入主畫面」
3. 桌面會出現「打卡」圖示，點開後全螢幕運作

## 功能

### 上班打卡

- 記錄打卡時間
- 自動計算預計下班時間（工時 7.5 小時，午休 12:00~13:00）
- 顯示加班起算時間（下班後 + 30 分鐘）
- 超過 09:30 顯示遲到提醒

### 下班打卡

- 記錄下班打卡時間

### 輸出範例

```
上班打卡：04/09 08:50
預計下班：17:20
加班起算：17:50
```

## 彈性上下班規則

| 條件 | 說明 |
|------|------|
| 08:30 前到班 | 以 08:30 起算，17:00 可下班 |
| 08:30 ~ 09:30 到班 | 依實際到班時間 + 7.5 小時計算 |
| 09:30 後到班 | 視為遲到，仍計算下班時間 |
| 加班 | 完成 7.5 小時工時後，休息 30 分鐘始得計算 |
